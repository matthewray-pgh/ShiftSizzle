# ShiftSizzle
workplace scheduling application and workforce management tooling

## Available Scripts

- `npm run dev`: Starts the Vite development server for client-side development with hot module replacement (HMR).
- `npm run server`: Runs the Express/Vite SSR server for server-side rendering and production-like behavior.

### Usage

- Use `npm run dev` for rapid client-side development and live updates.
- Use `npm run server` to run the SSR server and view the app as it will work in production.
