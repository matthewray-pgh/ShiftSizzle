
# ShiftSizzle
Workplace scheduling application and workforce management tooling

## Available Scripts

- `npm run dev`: Starts the Vite development server for client-side development with hot module replacement (HMR).
- `npm run build`: Creates the production bundle and mirrors it into `docs/`.
- `npm run test`: Runs the frontend smoke test suite with Vitest.

### Usage

- Use `npm run dev` for rapid client-side development and live updates. The app is now 100% client-side rendered and does not require any server-side rendering or custom server setup.

## Current Product Scope

- Dashboard with staffing metrics and schedule status.
- Team management with local persistence, filtering, editing, and archive flow.
- Scheduler with coverage targets, manual shift editing, auto-build, publish state, and schedule visualizations.
- Shifts and settings views backed by shared local application state.
